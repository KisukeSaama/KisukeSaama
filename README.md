## Hey, moi c'est Jonathan

Développeur web full-stack en formation à la Web@cadémie by EPITECH, je recherche une alternance de 14 mois à partir de septembre 2026.

Avant le développement, j'ai passé 3 ans comme référent informatique dans l'événementiel. Ce terrain m'a appris à diagnostiquer vite, prioriser sous pression et livrer des solutions fiables quand le contexte ne laisse pas beaucoup de marge.

Aujourd'hui, je construis des applications web, des APIs REST, des interfaces React et des outils qui mêlent code propre, logique produit et automatisation IA.
